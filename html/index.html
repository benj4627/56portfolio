FULD GRÆNSEFLADE - THE BOULEVARD TATTOO

The Boulevard Tattoo er en tatoveringsbutik med mange års erfaring, beliggende i Aalborg.

The Boulevard Tattoos eksisterende hjemmeside, optrådte mangelfuld på flere parametre, og var ikke blevet opdateret i omkring 10 år. Vores problemstilling beskriver hvordan man kan hæve både funktionalitet, brugervenlighed og design, samtidig med at vi matcher butikkens digitale tilstedeværelse, med den stemning, værdier og brand deres fysiske butik repræsenterer.

Vores mål var er omfavne overstående og levere en moderne, intuitiv og brugercentreret grænseflade, hvor fokus var lever et produkt, der stemte overens med butikkens overall branding og stil

  • HTML
  • CSS
  • JavaScript
  • UX/UI Design

WEBAPP - DEN BLÅ FESTIVAL

Den Blå Festival er en årlig jazz og blues festival, som finder sted i Aalborg i August

Vores problemstilling beskriver hvordan Den Blå Festivals program findes som trykt, fysisk program og som PDF format på den nuværende hjemmesiden, stort set en direkte kopi af det trykte.

Det er ikke en optimal løsning og begge mangler digital binding mellem koncerter og program. Derfor vil vi skabe en brugervenlig grænseflade, der hjælper brugerne med nemt og hurtigt at finde den information de søger, både før, under og efter festivalen.

  • HTML
  • CSS
  • JavaScript

KOMPONENT - NOTIFIKATIONSSIDE

Ud fra allerede eksisterende styleguide og high definition statisk mockup fra frontendmentor.io, realiseres her en notifikationsside, som skal ses som en del af et større website.

Her bliver der altså kun programmet denne specifikke del af den fulde side, med fokus på at skabe funktionalitet med vanilla JavaScript, semantisk og tilgængeligt HTML og genanvendelig CSS. Derudover kan det færdig produkt ses korrekt på både desktop og mobile viewport.
